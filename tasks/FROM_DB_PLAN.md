# Generata DDL Tabınbdaki "From Db" seçince izlenecek yol
1. Marta bağlanıp model açılmış olmalıdır.
2. FEModel_DB ile model import edilir(Daha önce bu başarılı bir şekilde yapılmıştı).
3. Erwindeki tablarda açık olan model olarak tekrar orjinal seçilir.
4. Daha önce bilinen "Review" butonu ekranı ile devam edilerek fark DDL'i oluşturulur.
5. "Review" den kastettiğim aynı ekrandaki "From Mart" seçip, önceki versiyonlarla ypaılan kıyaslama yapabilmeyi kastediyorum.
